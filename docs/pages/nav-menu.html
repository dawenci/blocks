API

属性名 类型 默认值 说明 必须
collapse boolean false 是否折叠（最小化）模式。
dark boolean 是否使用 dark 主题。
data 导航的数据，该属性为 prop，非 attr。接受一个 JavaScript 数组。数组中的项可以为菜单项，或者分组。其中菜单项可以有子菜单。 菜单项的数据结构： 
interface MenuItem {
  active?: boolean
  disabled?: boolean
  handler?: () => void
  href?: string
  icon?: string
  label: string
}
菜单分组的的数据结构: 
interface MenuGrup {
  title: string
  data: MenuItem[]
}
horizontal boolean 是否水平模式。
inline boolean （垂直模式专用）子菜单是否直接内联渲染（而非弹出菜单模式）。
size 'small'|'large' 菜单的尺寸。默认不设置为中等尺寸。
