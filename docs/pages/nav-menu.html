API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
collapse collapse boolean false 是否折叠（最小化）模式。
- data (MenuItem | MenuGroup)[] 
  interface MenuItem {
    active?: boolean
    disabled?: boolean
    handler?: () => void
    href?: string
    target?: string
    icon?: string
    label: string
    children: (MenuItem | MenuGroup)[]
  }
  
  interface MenuGroup {
    title: string
    data: MenuItem[]
  }
[] 导航的数据。接受一个 JavaScript 数组。数组中的项可以为菜单项，或者分组。其中菜单项可以有子菜单。
enter-delay enterDelay number 150 鼠标移上多少毫秒弹出子菜单。
horizontal horizontal boolean false 是否水平模式。
inline inline boolean false （垂直模式专用）子菜单是否直接内联渲染（而非弹出菜单模式）。
leave-delay leaveDelay number 200 鼠标移上多少毫秒弹出子菜单。
size size 'small' | 'large' 无 菜单的尺寸。默认不设置为中等尺寸。
