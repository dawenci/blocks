API

Attrs / Props

支持传入 date 组件的属性控制日期面板。

attr 名 prop 名 类型 默认值 说明
clearable clearable boolean false 是否可以清除值（显示清除按钮）。
placeholder-first placeholderFirst string '开始日期' 开始日期的 placeholder 文字内容。
placeholder-second placeholderSecond string '结束日期' 结束日期的 placeholder 文字内容。
- value [Date | null, Date | null] [null, null] 选中的日期区间值。

Events

事件名称 事件对象 说明
change CustomEvent<{ value: [Date | null, Date | null] }> 选中值变化时触发。
closed CustomEvent 日期面板关闭时触发。
opened CustomEvent 日期面板打开时触发。
