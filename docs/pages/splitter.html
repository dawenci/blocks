splitter

依赖引入

根据自己项目调整 src 的路径

默认使用

垂直方向

拖拽柄尺寸

最大最小尺寸限制

使用 max, min 属性，限制 pane 的尺寸调整幅度。

不自动分配

通过设置 grow、shrink 为 0，以阻止 pane 随着容器尺寸变化自动调整尺寸。

API

(bl-splitter) Attrs / Props

属性名 类型 默认值 说明 必须
direction 'horizontal' | 'vertical' 'horizontal' 面板排列方向，默认水平排列。
handle-size number 6 拖拽柄的尺寸。

(bl-splitter-pane) Attrs / Props

属性名 类型 默认值 说明 必须
basis number Pane 的基准尺寸。
grow number 1 Pane 自动拉伸尺寸的比例因子。
max number Infinity Pane 的最大尺寸限制。
min number 0 Pane 的最小尺寸限制。
shrink number 1 Pane 自动收缩尺寸的比例因子。

(bl-splitter) Events

事件名称 事件对象 说明
pane-resized { detail: { offset, $pane, $prevPane } } Pane 尺寸调整时触发。
pane-close { detail: { $pane } } Pane 完全折叠时触发。
pane-open { detail: { $pane } } Pane 从折叠状态展开时触发。
layout 重新计算布局时触发。
