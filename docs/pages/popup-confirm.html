默认使用

使用 `message` 属性，设置确认消息，使用 `icon` 属性，设置消息图标。

点击

按钮事件处理

设置 `confirm` 和 `cancel` 为函数，可以处理确定、取消按钮的事件。

如果 confirm 函数返回 Promise，则会进入 loading，等待 Promise 结束。

点击

API

属性名 类型 默认值 说明 必须
cancel () => void 取消按钮事件处理器。
confirm () => Promise | undefined 确认按钮事件处理器。
icon 图标。可以设置为已注册的 SVG 图标，参考图标说明（TODO）。
message string 确认信息文本。
