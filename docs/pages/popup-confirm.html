默认使用

使用 `message` 属性，设置确认消息，使用 `icon` 属性，设置消息图标。

点击

按钮事件处理

设置 `onConfirm` 和 `onCancel` 为函数，可以处理确定、取消按钮的事件。

如果 `onConfirm` 函数返回 Promise，则会进入 loading，等待 Promise 结束。

点击

API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
icon icon string 图标。可以设置为已注册的 SVG 图标，参考图标说明（TODO）。
message message string 确认信息文本。
open open boolean false popup 是否弹出。
- onCancel () => Promise<any> 取消按钮事件处理器。
- onConfirm () => Promise<any> 确认按钮事件处理器。

Methods

函数名称 类型 说明
confirm () => Promise<any> 提交确认
cancel () => Promise<any> 提交取消

Events

事件名称 事件对象 说明
cancel CustomEvent> 取消时触发。
confirm CustomEvent> 确认时触发。
