progress

依赖引入

根据自己项目调整 src 的路径

默认

显示百分比

使用 percentage 属性启用百分比显示

状态

可设置 'status' 属性切换状态，可选状态有 'success', 'error', 'warning'

API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
percentage percentage boolean false 是否显示百分比。
status status 
  | 'success'
  | 'error'
  | 'warning'
无 进度条状态。
value value number 0 当前百分比。
