API

Attrs / Props

属性名 类型 默认值 说明 必须
activable boolean false 是否可以高亮选中结点。
active-key string | number 当前高亮选中的结点的 key。
border boolean false 条目之间是否显示分割线。
check-on-click-node boolean false 是否在点击结点时选中。
check-strictly boolean false 多选时，父子结点是否独立选择而不联动。
data Array 数据源数组，子结点字段为 'children'。
default-fold-all boolean false 默认是否全部折叠。
disabled boolean false 是否不可用状态。
expand-on-click-node boolean false 是否在点击条目任意位置时折叠、展开子树。
id-field string 'id' 结点的唯一 key 生成字段（通常使用数据 id）。
indent-unit number 16 子结点的缩进距离（px）。
label-field string 'label' 结点的显示文本字段。
search string 搜索条目的关键词。
stripe boolean false 是否显示斑马纹背景。

Methods

函数名称 类型 说明
disableActiveMethod (data) => boolean 是否禁止条目被高亮选中的检测方法。
disableCheckMethod (data) => boolean 是否禁止条目被单选、复选选中的检测方法。
disableToggleMethod (data) => boolean 是否禁止条目被展开、折叠的检测方法。
keyMethod (data) => string 结点的唯一 key 生成方法，优先级高于 id-field 属性。
labelMethod (data) => string 结点内容的显示方法。优先级高于 label-field 属性。
