API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
checked checked boolean false 是否选中。
disabled disabled boolean false 是否禁用。
indeterminate indeterminate boolean false 是否半选中状态，如果指定了 checked，则 checked 优先。

事件名称 事件对象 说明

change CustomEvent<{ checked: boolean }> 选中状态切换时触发。
