Attrs / Props

attr 名 prop 名 类型 默认值 说明
clearable clearable boolean false 是否可以清除值（显示按钮）。
placeholder-first placeholderFirst string '开始日期时间' 开始日期时间的 placeholder 文字内容。
placeholder-second placeholderSecond string '结束日期时间' 结束日期时间的 placeholder 文字内容。
- value [Date | null, Date | null] [null, null] 选中的日期时间区间值。

Events

事件名称 事件对象 说明
change CustomEvent<{ value: [Date | null, Date | null] }> value 变化时触发。
opened 日期时间选择面板弹出后触发。
closed 日期时间选择面板关闭后触发。
click-clear 点击清除按钮时触发。
