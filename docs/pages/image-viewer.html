API

Attrs / Props

属性名 类型 默认值 说明 必须
alt string img 的 alt。
fallback string 载入失败时显示的图片 url。
fit 'fill' | 'contain' | 'cover' | 'none' | 'scale-down' 图片如何适应容器空间。
manual boolean false 是否使用手动加载（手动调用 load()，而非自动加载）。
placeholder string 占位图片 url。
src string 图片的 url。

Events

事件名称 事件参数 说明
error 加载错误时触发。
loaded 加载完成时触发。
loading 加载时触发。
