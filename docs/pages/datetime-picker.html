API

属性名 类型 默认值 说明 必须
clearable boolean false 是否可以清除值（显示按钮）。
prefix-icon string 前缀图标名称，可以设置为已注册的 SVG 图标，参考图标说明（TODO）。
readonly 是否只读。
size 'small'|'large' 尺寸。默认不设置为中等尺寸。
suffix-icon string 后缀图标名称，可以设置为已注册的 SVG 图标，参考图标说明（TODO）。
value null | Date 读写组件 model，null 表示未选中，Date 为单选的值读写。

Events

事件名称 事件对象 说明
change CustomEvent<{ value: null | Date }> value 变化时触发。值清空时，CustomEvent 的 detail.value 为 `null`。
opened 日期时间选择面板弹出后触发。
closed 日期时间选择面板关闭后触发。
click-clear 点击清除按钮时触发。
