API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
clearable clearable boolean false 是否可以清除值（显示按钮）。
format format string 'YYYY-MM-DD HH:mm:ss' 选择结果内容格式化方式。
open open boolean false 日期时间选择面板是否弹出展示。
placeholder placeholder string '请选择日期时间' placeholder 文字内容。
disabledDate 日期面板的按钮 disabled 状态控制函数，参考 date 组件文档。
disabledTime 时间面板的按钮 disabled 状态控制函数，参考 time 组件文档。
- value Date | null null 选中的日期值

Events

事件名称 事件对象 说明
change CustomEvent<{ value: null | Date }> value 变化时触发。
closed CustomEvent 日期时间选择面板关闭后触发。
opened CustomEvent 日期时间选择面板弹出后触发。
click-clear CustomEvent 点击清除按钮时触发。
