API

Attrs / Props

支持传入 time 组件的属性控制日期面板。

attr 名 prop 名 类型 默认值 说明
clearable clearable boolean false 是否可以清除值（显示清除按钮）。
- hour number(0 - 23) 时
- minute number(0 - 59) 分
open open boolean false 时间面板是否展示。
- second number(0 - 59) 秒
- value { hour: number; minute: number; second: number } | null null model 值

Events

事件名称 事件对象 说明
change CustomEvent<{ hour: number; minute: number; second: number }> 选中的时间变化时触发。
