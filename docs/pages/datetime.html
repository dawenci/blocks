基础使用

面板起始深度

'start-depth' 设置为初始化时展示的面板深度，支持月、年、年代、世纪。默认等于 `depth` 的取值

面板最上层深度

'min-depth' 设置面板最多可以上卷到哪个层级的深度，支持月、年、年代、世纪。默认为 `century`

星期的起始

可以通过 'start-week-on' 设置每周的第一天为星期几。

API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
disabled disabled boolean false 是否禁用。
depth depth 参考 date 组件文档
disabledDate 参考 date 组件文档
format format string 'YYYY-MM-DD HH:mm:ss' 派发 `select-list:change` 事件时，label 的格式化方式。
time-format timeFormat string 'HH:mm:ss' 选中时间时，显示在顶部的选中值格式化方式。
min-depth minMepth 参考 date 组件文档
start-depth startDepth 参考 date 组件文档
start-week-on startWeekOn 参考 date 组件文档
- selected Date | null null 当前选中的日期值

Events

事件名称 事件对象 说明
change CustomEvent<{ value: Date | null }> 日期时间变化时触发。
select-list:change CustomEvent<{ value: ISelected[] }>
type ISelected = { value: T; label: string } 		日期时间变化时触发
