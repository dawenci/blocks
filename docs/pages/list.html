API

Attrs / Props

属性名 类型 默认值 说明 必须
border boolean false 是否启用项描边。
data Object[] 列表数据（对象列表），该属性是 prop，非 attr。
disabled boolean false 是否禁用。
disabled-label string 'disabled' 项禁用的依据字段名称。
id-field string 'id' 数据的唯一 key 字段名称。
label-field string 'label' 数据的 label 字段名称。
multiple boolean false 是否多选模式
checked 选中的 id 或 id 数组（多选模式）。该属性是 prop，非 attr。
search string 搜索条目的关键词。
stripe boolean false 是否启用斑马纹。

Methods

函数名称 类型 说明
keyMethod (data) => string 结点的唯一 key 生成方法，优先级高于 id-field 属性。
labelMethod (data) => string 结点内容的显示方法。优先级高于 label-field 属性。
