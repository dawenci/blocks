API

属性名 类型 默认值 说明 必须
capturefocus boolean false 是否捕获焦点。如果是，tab 键不会将焦点移出 Drawer。
close-on-click-outside boolean false 是否在点击外面的时候，自动关闭 Drawer。
close-on-escape boolean false 是否在按下 Esc 时，自动关闭 Drawer。
dark boolean false 是否使用 dark 主题。
mask boolean false 是否显示遮罩。
open boolean false 抽屉的打开状态。
placement 'top' | 'right' | 'bottom' | 'left' 'right' Drawer 的出现位置。默认从右往左拉出。
size string 30% Drawer 的尺寸，用作 width/height 的 style 值。
title string 抽屉的标题
