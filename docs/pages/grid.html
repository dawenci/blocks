默认栅格

24 列栅格系统

主轴方向排版

bl-row 的 justify 属性可以设置排版方式。

对齐

bl-row 的 align 属性可以设置对齐方式。

间隔

bl-row 的 gutter 属性可以设置列间隔。

换行

bl-row 的 wrap 属性可以设置换行。

布局偏移

offset

属性 offset 可以设置左侧偏移，以间隔出一块区域。

pull, push

属性 pull 可以将自身往左边移动若干栅格列宽度的举例，而 pull 则往右移动。

API

(row) Attrs / Props

属性名 类型 默认值 说明 必须
align 'top' | 'middle' | 'bottom' 'top' 垂直方向如何对齐元素。
gutter number 0 栅格之间的间隙尺寸（px）。
justify 'start' | 'end' | 'center' | 'space-around' | 'space-between' 'start' 水平方向如何分配元素周围的空间。
wrap boolean false 是否允许换行。

(col) Attrs / Props

属性名 类型 默认值 说明 必须
offset number 0 左侧空出多少个栅格列。
pull number 0 往左移动多少个栅格列。
push number 0 往右移动多少个栅格列。
span number 24 尺寸横跨多少个栅格列。
