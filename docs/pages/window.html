API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
actions actions string 'minimize,maximize,close' 窗口动作按钮，使用字符串 'minimize', 'maximize', 'close' 任意组合，使用英文逗号分割，顺序不限。
capturefocus capturefocus boolean false 是否将焦点限制在窗口里面，而不被 tab 切走。
icon icon string 标题栏图标。
maximized maximized boolean false 是否最大化窗口。
minimized minimized boolean false 是否最小化窗口。
name name string 窗口的标题。
open open boolean false 窗口是否打开。

Events

事件名称 事件对象 说明
bl:resize CustomEvent<{ width: number; height: number }> 窗口尺寸变化触发。
closed 窗口关闭之后（动画播放完毕）触发。
opened 窗口打开之后（动画播放完毕）触发。
