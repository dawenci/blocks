API

Attrs / Props

属性名 类型 默认值 说明 必须
actions string 'minimize,maximize,close' 窗口动作按钮，使用字符串 'minimize', 'maximize', 'close' 任意组合，使用英文逗号分割，顺序不限。
capturefocus boolean false 是否将焦点限制在窗口里面，而不被 tab 切走。
dark boolean false 是否 dark 模式。
icon string 标题栏图标。
maximized boolean false 是否最大化窗口。
minimized boolean false 是否最小化窗口。
name string 窗口的标题。
open boolean false 窗口是否打开。

Events

事件名称 事件参数 说明
close 窗口关闭之后（动画播放完毕）触发。
open 窗口打开之后（动画播放完毕）触发。
resize { detail: { data } } 窗口尺寸变化触发。
