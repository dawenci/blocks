API

Attrs / Props

支持传入 date 组件的属性控制日期面板。

attr 名 prop 名 类型 默认值 说明
clearable clearable boolean false 是否可以清除值（显示清除按钮）。
open open boolean false 日期选择面板是否弹出展示。
size size 'small' | 'large' 无 尺寸
- value 单选: Date | null
多选: Date[] 		单选: null
多选: [] 		选中的日期值

Events

事件名称 事件对象 说明
change 单选: CustomEvent<{ value: Date | null }>
多选: CustomEvent<{ value: Date[] }> 		日期变化时触发。
closed CustomEvent 日期面板关闭时触发。
opened CustomEvent 日期面板打开时触发。
