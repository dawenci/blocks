API

Attrs / Props

属性名 类型 默认值 说明 必须
clearable boolean false 是否可以清除。
multiple boolean false 是否多结果模式。
multiple-mode 'plain' | 'tag' 'plain' 多结果显示模式。
prefix-icon string 前缀图标。
searchable boolean false 是否可以搜索。
size 'small' | 'large' 尺寸。默认不设置为中等尺寸。
suffix-icon string 后缀图标。
tag-clearable boolean false 结果 tag 是否渲染关闭按钮。

Events

事件名称 事件参数 说明
search 搜索框录入时触发。
click-prefix-icon 点击前缀图标时触发。
click-suffix-icon 点击后缀图标时触发。
click-clear 点击清除按钮时触发。
