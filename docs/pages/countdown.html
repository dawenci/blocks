依赖引入

根据自己项目调整 src 的路径

常规使用

value 表示截止时间戳

format 格式

可以通过 `format` 属性设置格式化输出。

自定义样式

CSS 中可以通过 `::part()` 分别控制 `day`, `hour`, `minute`, `second`, `millisecond`, `text` 控制各个部分。

API

属性名 类型 默认值 说明 必须
format string 'H:mm:ss' 日期格式化方式，支持：
  • `D`（天）
  • `H`（时）
  • `HH`（时，自动补零）
  • `m`（分）
  • `mm`（分，自动补零）
  • `s`（秒）
  • `ss`（秒，自动补零）
  • `SSS`（毫秒）
value string 时间戳 Y
