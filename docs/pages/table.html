简单表格

使用 columns 配置列，并传入 data 数据，即可渲染表格。

边框

使用 border 属性启用边框模式。table 将会渲染外边框，以及单元格之间的垂直分割线。

嵌套表头

通过 children 递归设置嵌套表头。

固定列

任意列启用了 fixedLeft 或 fixedRight 后，即会渲染为固定列。在嵌套的子列中设置固定列时，会上溯到根列，全部编程固定列。

合计行

任一列的配置中，出现 summaryRender 函数，即自动启用合计行渲染。

可调整列宽

列配置中，启用 resizable，即表示允许拖拽调整列宽。

API

属性名 类型 默认值 说明 必须
border boolean false 是否渲染边框和单元格垂直分割线
columns Array<ColumnSetting> 表格列配置。
ColumnSetting 的可配置项如下： 
label: 表头文字。


headRender：表头自定义渲染函数。函数类型：(column) => string | number | Node。


render：单元格内容自定义渲染函数。函数类型：(data, column, $cell) => string | number | Node。


prop：单元格内容对应的数据字段。


align：水平对齐方式。


width：列宽。


minWidth：最小列宽。


maxWidth：最大列宽。


fixedLeft：是否左固定列。


fixedRight：是否右固定列。


sortOrder：排序顺序（未实现），可选值为 'none', 'ascending', 'descending'。


resizable：是否可以调整尺寸（未实现）。


summaryRender：合计行单元格内容渲染方法。函数类型：(column, index, data, $cell) => string | number | Node。


children：嵌套列定义。
data Array 表格行数据。默认渲染规则为，每一行对应一个 JavaScript 对象，每一列对应一个字段（通过 columns 配置渲染的字段名）。
