API

属性名 类型 默认值 说明 必须
border boolean false 是否渲染边框和单元格垂直分割线
columns Array<ColumnSetting> 表格列配置。
ColumnSetting 的可配置项如下： 
label: 表头文字。


headRender：表头自定义渲染函数。函数类型：(column) => Node。


render：单元格内容自定义渲染函数。函数类型：(data, column, $cell) => Node。


prop：单元格内容对应的数据字段。


align：水平对齐方式。


width：列宽。


minWidth：最小列宽。


maxWidth：最大列宽。


fixedLeft：是否左固定列。


fixedRight：是否右固定列。


sortOrder：排序顺序（未实现），可选值为 'none', 'ascending', 'descending'。


resizable：是否可以调整尺寸（未实现）。


summaryMethod：合计方法（未实现）。


children：嵌套列定义。
data Array 表格行数据。默认渲染规则为，每一行对应一个 JavaScript 对象，每一列对应一个字段（通过 columns 配置渲染的字段名）。
