尺寸设置

可通过 size 设置尺寸，可选值: `small`, `large`

关闭按钮

设置 `closeable` 属性，则启用关闭按钮，点击会触发 `close` 的 DOM 自定义事件

主题

type

`type` 可以设定主题类别，支持 `primary`, `danger`, `warning`, `success`

outline

支持 `outline` 变种样式

圆角

可通过 round 设置圆角

API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
closeable closeable boolean false 是否显示关闭按钮。
outline outline boolean false 是否 outline 风格。
round round boolean false 是否圆角风格。
size size 'small' | 'large' 无 默认中等尺寸。
type type 'primary' | 'danger' | 'warning' | 'success' 无
