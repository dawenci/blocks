API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
capturefocus capturefocus boolean false 是否捕获焦点。如果是，tab 键不会将焦点移出 Dialog。
closeable closeable boolean false 是否提供关闭按钮。
close-on-click-mask closeOnClickMask boolean false 点击遮罩时关闭。
close-on-click-outside closeOnClickOutside boolean false 点击对话框外部时关闭。
close-on-press-escape closeOnPressEscape boolean false 按下 Esc 时关闭。
mask mask boolean false 是否显示遮罩。
open open boolean false 对话框的打开状态。
title-text titleText string 无 对话框的标题。
unmount-on-closed unmountOnClosed boolean false 关闭时自动卸载 DOM。
