基础单选

多选

'mode' 设置为 'multiple' 可以启用多选模式，可以用 'max' 属性限制最多选择的日期数量。

区间

'mode' 设置为 'range' 可以启用区间模式。

选择深度

'depth' 设置为选择的深度，支持月、年、年代。

此外，可以设置初始化显示的面板深度（startdepth，默认等于 depth），以及往上最多可以切换到多浅的深度（mindepth：默认世纪，century）

星期的起始

可以通过 'start-week-on' 设置每周的第一天为星期几。

API

Attrs / Props

属性名 类型 默认值 说明 必须
depth 'month' | 'year' | 'decade' 'month' 选择值的深度。month 代表最深按照月展示选项，可以选择到 “天”；year 代表最深按照年份展示选项，可以选择到 “月”；decade 代表最深按照十年展示选项，可以选择到 “年”。
disableMethod (data: { year: number, month?: number, date?: number }) => boolean 校验（depth 末级）选项是否禁用的检查方法。该属性无对应 attr，只能作为 prop 传入。
loading boolean false 是否进入 loading 状态（遮罩）。
max number 多选模式的话，最多能选择多少个值。默认不设置表示不限。
mindepth 'month' | 'year' | 'decade' | 'century' 'century' 可切换至的最小深度。
mode 'single' | 'multiple' | 'range' 'single' 选择模式。支持单选、多选、区间选择。
startdepth 初始化显示的面板深度。
start-week-on 0 | 1 | 2 | 3 | 4 | 5 | 6 1 每周从星期几开始，0 代表星期天，1 代表星期一，顺序类推。对应的 prop 为 `startWeekOn`
value model 值。

Events

事件名称 事件参数 说明
input { detail: { value } } value 变化时触发。
next-century { detail: { century }} 面板切换到下个世纪时触发。
next-decade { detail: { century, decade } } 面板切换到下个十年时触发。
next-month { detail: { century, decade, year, month } } 面板切换到下月时触发。
next-year { detail: { century, decade, year } } 面板切换到下一年时触发。
panel-change { detail: { viewDepth } } 面板层级变化时触发。
prev-century { detail: { century }} 面板切换到上个世纪时触发。
prev-decade { detail: { century, decade } } 面板切换到上个十年时触发。
prev-month { detail: { century, decade, year, month } } 面板切换到上月时触发。
prev-year { detail: { century, decade, year } } 面板切换到上一年时触发。
