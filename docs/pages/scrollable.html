scrollable

依赖引入

根据自己项目调整 src 的路径

默认

滚动阴影效果

API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
shadow shadow boolean false 是否在边沿显示可滚动提示阴影

Events

事件名称 事件对象 说明
bl:resize CustomEvent<{ width: number; height: number }> 尺寸变化时触发。
bl:scroll CustomEvent 发生滚动时触发。

CSS Variables

变量名 默认值 说明
--bg-thumb black 滚动条滑块的背景颜色（设置不透明的颜色即可，显示时会自动设置透明度）。
--bg-track transparent 滚动条滑轨的背景颜色。
