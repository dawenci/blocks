API

Attrs / Props

属性名 类型 默认值 说明 必须
hex string prop，获取、设置十六进制颜色值。
hsl [number, number, number] prop，获取、设置 hsl 颜色。
hsla [number, number, number, number] prop，获取、设置 hsla 颜色。
hsv [number, number, number] prop，获取、设置 hsv 颜色。
hsva [number, number, number, number] prop，获取、设置 hsva 颜色。
rgb [number, number, number] prop，获取、设置 rgb 颜色。
rgba [number, number, number, number] prop，获取、设置 rgba 颜色。
value number attr / prop，获取、设置颜色的 rgba 数值。
32 位整数，每 8 位存储一个通道，从高到低位，按照 a,b,g,r 通道的顺序存储。

Methods

函数名称 类型 说明
format (fmt: 'rgba' | 'rgb' | 'hex' | '#rgb' | '#rrggbb' | '#rrggbbaa' | 'hsl' | 'hsla' | 'hsv' | 'hsva') => string 返回格式化后的颜色值。

Events

事件名称 事件对象 说明
bl:color:change CustomEvent<{ value: number }> 颜色变化时触发。
change CustomEvent<{ value: number }> 颜色变化时触发。
