API

Attrs / Props

属性名 类型 默认值 说明 必须
hex string prop，获取、设置十六进制颜色值。
hsl [number, number, number] prop，获取、设置 hsl 颜色。
hsla [number, number, number, number] prop，获取、设置 hsla 颜色。
hsv [number, number, number] prop，获取、设置 hsv 颜色。
hsva [number, number, number, number] prop，获取、设置 hsva 颜色。
rgb [number, number, number] prop，获取、设置 rgb 颜色。
rgba [number, number, number, number] prop，获取、设置 rgba 颜色。
value string attr / prop，获取、设置颜色。支持的格式：
  • 十六进制，例如：`#f00`, `#ff0000`, `#ff0000ff`
  • rgb，例如：`rgb(255,0,0)`
  • rgba，例如：`rgba(255,0,0,1)`
  • hsl，例如：`hsl(0,100%,50%)`
  • hsla，例如：`hsla(0,100%,50%,1)`
  • hsv，例如：`hsv(0,100%,100%)`
  • hsva，例如：`hsva(0,100%,100%,1)`

Methods

函数名称 类型 说明
toHexString () => string 返回十六进制颜色值。
toRgbString () => string 返回 rgb 颜色值。
toRgbaString () => string 返回 rgba 颜色值。
toHslString () => string 返回 hsl 颜色值。
toHslaString () => string 返回 hsla 颜色值。
toHsvString () => string 返回 hsv 颜色值。
toHsvaString () => string 返回 hsva 颜色值。

Events

事件名称 事件参数 说明
change 颜色变化时触发。
