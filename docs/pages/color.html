API

Attrs / Props

attr 名 prop 名 类型 默认值 说明
- hex string 获取、设置十六进制颜色值。
- hsl [number, number, number] [0, 1, 0.5] 获取、设置 hsl 颜色，3 位分别对应 hue(0-360), saturation(0-1), lightness(0-1)。
- hsla [number, number, number, number] [0, 1, 0.5, 1] 获取、设置 hsla 颜色，4 位分别对应 hue(0-360), saturation(0-1), lightness(0-1), alpha(0-1)。
- hsv [number, number, number] [0, 1, 1] 获取、设置 hsv 颜色，3 位分别对应 hue(0-360), saturation(0-1), value(0-1)。
- hsva [number, number, number, number] [0, 1, 1, 1] 获取、设置 hsva 颜色，4 位分别对应 hue(0-360), saturation(0-1), value(0-1), alpha(0-1)。
- rgb [number, number, number] [255,0,0] 获取、设置 rgb 颜色，3 位分别对应 red(0-255), green(0-255), blue(0-255)。
- rgba [number, number, number, number] [255,0,0,1] 获取、设置 rgba 颜色，4 位分别对应 red(0-255), green(0-255), blue(0-255), alpha(0-1)。
value value number 4278190335 获取、设置颜色的 rgba 数值。
32 位整数，每 8 位存储一个通道，从高到低位，按照 a,b,g,r 通道的顺序存储。
