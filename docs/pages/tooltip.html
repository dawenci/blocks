Attrs / Props

属性名 类型 默认值 说明 必须
close-delay number 200 （hover 模式）鼠标移走延迟关闭毫秒数。
content string 提示内容。
open-delay number 200 （hover 模式）鼠标移上去延迟打开毫秒数。
trigger-mode 'click' | 'hover' 'hover' 触发模式
